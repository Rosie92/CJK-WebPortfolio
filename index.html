#강아지는_훌륭하다 #견주들을_위한_공간

프로젝트 설명

강아지는 훌륭하다는 점차 늘어나는 반려인들을 위한 커뮤니티 웹사이트이다.
한국농촌경제연구원(KREI)의 조사에 따르면 한국 29.4%의 가구가 반려동물을 키운다고 발표하였으며, 이를통해 반려인들을 위한 공간을 만들고자 하였다.
주요 기능으로는 카카오 로그인을 구현하여 번거로운 회원가입 절차를 간소화 하였으며 회원 정보 또한 카카오와 연동하여 카카오의 정보만 확인 및 변경한다.
그리고 크롤링 기술을 통하여 견종백과와 실시간 동물 관련 뉴스를 제공하며, 혼자사는 직장인 또는 학생들을 위해 카카오 맵을 활용하여 지역별 강아지 유치원 위치 정보를 제공하고, 반려견 산책시 중요사항인 날씨 확인을 위해 오픈웨더API를 활용해 기상 정보를 실시간으로 서비스 가장 상단에 제공하고 있다.
또한 게시판과 실시간 채팅 기능을 추가하여 사용자들간의 정보교환 활성화를 유도하고 있다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : MySQL, MongoDB
- 적용 데이터 : HappyPet 뉴스정보, BeMyPet 견종백과
- 적용 API : KakaoLogin API (RestAPI), KakaoMap API, OpenWeather API
- 기타 : JavaScript, Crawling, jQuery, Ajax, HTML5, Bootstrap
- 클라우드 : AWS EC2

#BandMoA #밴드모아

프로젝트 설명

"밴드모아"는 전국의 모든 뮤지션들을 위한 커뮤니티 공간이다.
생업과 취미 여부를 떠나 학생, 직장인, 남녀노소 구분 없이 음악을 사랑하는 모든 사람들이 의견을 나누고 정보를 공유할 수 있는 공간을 만들고자 하였다.

적용기술

- 프레임워크 : Spring Framework
- 데이터베이스 : Oracle 11g
- 적용 공공데이터(OpenAPI) : 서울시 공연장 정보
- 기타 : KakaoMap API, JS, jQuery, Ajax, HTML5, Bootstrap
